## Política de versão (projeto)

Comando: `dart run tool/bump_version.dart`

- Cada release: **versionName** sobe um passo (`1.2.5` → `1.2.6` → … → `1.2.9` → `1.3.0`).
- **versionCode** (+N no pubspec) sobe **1** a cada bump (obrigatório na Play Store).
- Telas **Entrar** e **Sobre** leem a versão do build (`PackageInfo`); não editar texto fixo nelas.
